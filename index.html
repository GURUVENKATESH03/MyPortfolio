Project 6: Speech-to-Text Web Application

Overview

A real-time speech-to-text web application that converts spoken words into written text using the Web Speech API. It supports multiple languages, displays interim results while listening, and allows users to download or clear the transcribed text. This tool enhances accessibility, productivity, and voice-driven interaction directly in the browser.

Tech Stack

  • Frontend: HTML, CSS, JavaScript (Vanilla)
  • API: Web Speech API (SpeechRecognition)
  • Browser Support: Chrome, Edge (with `webkitSpeechRecognition` fallback)

Key Features

  • Start and stop live speech recognition with a toggle button
  • Displays interim transcription while speaking
  • Final transcribed text is appended to the result area
  • Supports multiple languages selectable from a dropdown
  • Download the transcribed text as a `.txt` file
  • Clear button to reset the result area
  • Handles various errors: no speech, mic not found, permission denied, etc.

Contributions

  • Implemented live speech recognition with proper error handling
  • Built language selection logic using dynamic option population
  • Added support for interim and final transcript handling
  • Integrated download and clear functionality for usability
  • Designed user feedback via UI updates and alerts

Project 5: Voice enabled Text Reader for Visually Impaired

Overview

Our system offers a voice-activated, completely offline text-to-speech framework developed for the Raspberry Pi 5 to address the accessibility challenges faced by visually impaired individuals. It captures text from printed documents or images and reads it aloud using natural speech synthesis, significantly improving access to printed content.

Tech Stack

  • Language: Python
  • OCR: EasyOCR for multilingual recognition (English, Kannada, Telugu)
  • Image Preprocessing: CLAHE (Contrast Enhancement), Median Blurring (Noise Reduction)
  • Text Refinement: SymSpell (Spell Correction), IndicNLP (Text Normalization)
  • Transliteration: Rule-based Multilingual Attentive Transliteration System
  • Text-to-Speech: pyttsx3 (Offline Speech Synthesis)
  • Hardware: Raspberry Pi 5 (for portable offline deployment)

Key Features

  • Capture or upload images containing text
  • Multilingual OCR for English, Kannada, and Telugu using EasyOCR
  • Enhanced image preprocessing for better accuracy under poor lighting
  • Text refinement through spell correction and normalization
  • Rule-based transliteration for phonetic clarity of Indic scripts
  • Offline text-to-speech conversion using pyttsx3
  • Real-time audio output with voice customization (pitch/speed)
  • Fully offline support, ideal for remote and low-connectivity environments

Contributions

  • Developed the full OCR-to-speech pipeline with multilingual support
  • Implemented image preprocessing (CLAHE, blurring) for OCR enhancement
  • Integrated SymSpell and IndicNLP for text correction and normalization
  • Configured a rule-based transliteration model for clearer speech output
  • Integrated pyttsx3 for smooth, offline voice synthesis
  • Optimized the solution to run efficiently on Raspberry Pi 5 hardware
